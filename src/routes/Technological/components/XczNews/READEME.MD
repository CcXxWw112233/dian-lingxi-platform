# 头部导航高亮的效果

- 思考：是否要将数据写死，还是做循环遍历的形式？
- 写死：无法获取到当前点击的对象
- 循环：可以判断当前点击的path与location中的pathname是否匹配，匹配了，就给你加上高亮的类
